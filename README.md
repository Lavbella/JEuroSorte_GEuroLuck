## Português

###  Sobre o Projeto
O **JEuroSorte** é uma ferramenta avançada para Windows e WebMobile concebida para otimizar a escolha de chaves para o EuroMilhões. Ao contrário dos geradores tradicionais, o sistema afasta-se do acaso puro, cruzando **Análise Estatística** rigorosa com conceitos de **Numerologia** para reduzir de forma inteligente o universo de 139.838.160 combinações possíveis.

### A Abordagem Científica e Mística
A aplicação utiliza algoritmos proprietários que correm em menos de 60 segundos para classificar e ordenar chaves com base em dois pilares:
1. **Análise Estatística Descritiva:** Filtra e elimina grupos de chaves matematicamente improváveis (ex: 5 números seguidos ou chaves repetidas), gerando cerca de 40 gráficos comportamentais do histórico de sorteios.
2. **Algoritmos Baseados em Numerologia:** Permite a personalização da inteligência de cálculo através do mapeamento e cruzamento com os números e estrelas favoritos do utilizador.

###  Funcionalidades Principais
* **Filtros Avançados:** Algoritmos de pesquisa, ordenação, classificação e gestão de memória ultra-rápidos.
* **Painel Estatístico:** Cerca de 40 gráficos interativos para analisar a ligação e frequência entre números/estrelas.
* **Atualização do Histórico:** Após efetuar o registo na aplicação, terá acesso total para inserir as novas chaves premiadas e manter as estatísticas e gráficos 100% atualizados.
* **Totalmente Anónimo:** Registo encriptado sem recolha de dados pessoais ou utilização de cookies.
* **Nota de Ativação:** A antiga funcionalidade de ativação periódica foi **descontinuada**. A aplicação já não necessita de ativação para funcionar.
* **Nota de Segurança:** As credenciais criadas no registo anónimo servem apenas para o seu acesso seguro e não podem ser recuperadas após fechar a janela. Anote-as imediatamente.

## English

###  About the Project
**JEuroSorte** is an advanced decision-support tool available for Windows and WebMobile, designed to optimize EuroMillions lottery picks. Unlike traditional quick-picks, this system moves away from pure randomness by combining rigorous **Statistical Analysis** with **Numerology** to intelligently narrow down the massive pool of 139,838,160 possible combinations.

###  The Statistical & Numerological Approach
The application uses proprietary algorithms that process data in under 60 seconds, sorting combinations based on two main pillars:
1. **Descriptive Statistical Analysis:** Filters out mathematically improbable combinations (e.g., 5 consecutive numbers) and generates around 40 dynamic charts tracking historical draw behaviors.
2. **Numerology-Based Algorithms:** Customizes calculations by mapping and cross-referencing data with the user's specific favorite numbers and lucky stars.

###  Key Features
* **Advanced Filters:** Ultra-fast search, sorting, classification, and memory management algorithms.
* **Statistical Dashboard:** Around 40 interactive charts analyzing connections and frequencies between numbers/stars.
* **Database Updates:** Once registered, you can freely insert new winning keys to keep all statistical projections and charts 100% accurate and up to date.
* **Fully Anonymous:** Encrypted registration with zero PII (Personally Identifiable Information) collection and no cookies.
* **Activation Notice:** The legacy periodic activation feature has been **deprecated**. The application no longer requires activation to function.
* **Security Notice:** Credentials generated during the anonymous registration are solely for your secure access and cannot be recovered once the window is closed. Write them down immediately.
